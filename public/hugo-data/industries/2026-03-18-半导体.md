# 半导体行业研究报告
**日期：2026-03-18**

---

## 一、行业定义与概况

半导体是指常温下导电性能介于导体与绝缘体之间的材料，通常由硅、锗等元素构成。半导体行业是现代信息技术产业的核心，涵盖**集成电路(IC)设计、制造、封测**以及设备和材料等细分领域。

从产业链位置来看，半导体行业处于**电子信息产业的上游**，是智能手机、电脑、汽车电子、物联网等终端产品的核心支撑。

### 行业发展历程

| 阶段 | 时间 | 特征 |
|------|------|------|
| 萌芽期 | 1947-1970s | 晶体管诞生，集成电路初步发展 |
| 成长期 | 1980-2000s | PC时代推动，摩尔定律主导 |
| 爆发期 | 2010-2020s | 移动互联网、AI、5G驱动 |
| 自主可控期 | 2020-至今 | 国产替代加速，政策强力扶持 |

---

## 二、宏观环境分析（PEST）

### 1. 政策环境（Policy）

- **国产替代战略**：国家集成电路产业投资基金（大基金）累计投资超5000亿元，支持半导体制造、设备、材料等环节
- **扶持政策**：《新时期促进集成电路产业和软件产业发展的若干政策》（国发〔2020〕8号）提供财税、投融资等全方位支持
- **出口管制**：美国对华半导体出口限制升级，，倒逼国产替代加速
- **十四五规划**：明确提出突破先进制程、高端芯片等关键技术

### 2. 经济环境（Economy）

- **全球市场规模**：2025年全球半导体市场规模约6000亿美元，中国占比约35%
- **国产替代空间**：中国半导体设备国产化率不足20%，材料国产化率约30%，替代空间巨大
- **AI驱动**：ChatGPT引发AI算力需求爆发，GPU/HBM等高性能芯片需求激增

### 3. 社会环境（Society）

- **消费电子迭代**：智能手机、汽车智能化、可穿戴设备需求持续增长
- **数字化转型**：企业数字化、工业互联网带来增量需求
- **人才缺口**：中国半导体人才缺口约30万人，制约行业发展

### 4. 技术环境（Technology）

- **制程工艺**：台积电3nm量产，中芯国际14nm成熟，先进制程差距仍存
- **先进封装**：Chiplet、2.5D/3D封装成为延续摩尔定律的重要路径
- **第三代半导体**：SiC、GaN在新能源汽车、光伏等领域加速渗透

## 三、产业链全景

### 产业链结构

```
上游：EDA软件、设备（光刻机、刻蚀机、沉积设备）、材料（硅片、光刻胶、电子特气）
    ↓
中游：IC设计（CPU、GPU、FPGA、模拟芯片、存储芯片）→ IC制造（晶圆代工）→ IC封测
    ↓
下游：智能手机、汽车电子、计算机、通信设备、物联网、消费电子
```

### 价值链分布

| 环节 | 特点 | 利润率 | 话语权 |
|------|------|--------|--------|
| IC设计 | 技术密集，轻资产 | 20-40% | 高 |
| IC制造 | 资本密集，重资产 | 15-25% | 高 |
| IC封测 | 劳动密集，成熟工艺 | 8-15% | 中 |
| 设备 | 技术壁垒极高 | 25-40% | 极高 |
| 材料 |种类繁多，差异大 | 10-20% | 中 |

## 四、市场规模与趋势

### 全球市场

- **2025年市场规模**：约6000亿美元，同比增长约10%
- **2026年预测**：受益于AI算力需求爆发，预计同比增长12-15%，规模达6700亿美元

### 中国市场

- **2025年市场规模**：约2100亿美元（占全球35%），同比增长约15%
- **国产化率**：整体约25%，设备约15%，材料约30%，CPU/GPU等高端芯片不足10%

### 周期位置

当前半导体行业处于**库存周期底部+AI需求爆发**的叠加期：
- **存储芯片**：价格从2023年低点回升，2025年量价齐升
- **AI芯片**：供需两旺，GPU/HBM严重供不应求
- **成熟制程**：产能利用率逐步回升，车规芯片需求旺盛

## 五、竞争格局分析

### 全球竞争格局

| 领域 | 龙头企业 | 市场地位 |
|------|----------|----------|
| IC设计 | 英伟达、高通、AMD、博通 | 美国主导 |
| IC制造 | 台积电、三星、英特尔 | 台积电一家独大（60%份额） |
| 设备 | ASML、应用材料、泛林、东京精密 | 美国+荷兰主导 |
| 存储 | 三星、SK海力士、美光 | 韩国主导 |

### 中国竞争格局

**设计环节**：华为海思（受阻）、寒武纪、地平线、兆易创新、汇顶科技
**制造环节**：中芯国际（14nm成熟）、华虹半导体（成熟制程）
**封测环节**：长电科技、华天科技、通富微电（全球前三）
**设备环节**：北方华创、中微公司、拓荆科技（国产替代先锋）

### 五力模型分析

1. **供应商议价权**：高（设备/材料被少数外企垄断）
2. **客户议价权**：中（大型终端厂商议价能力强）
3. **新进入者威胁**：低（高壁垒、重资本）
4. **替代品威胁**：低（半导体是电子系统核心）
5. **现有竞争**：高（全球充分竞争）

## 七、政策环境

### 核心政策

1. **大基金三期**：注册资本3440亿元，重点投向AI芯片、HBM等先进领域
2. **国产替代专项**：半导体设备、材料进口关税减免
3. **车规芯片扶持**：支持整车厂采购国产芯片
4. **人才政策**：集成电路专业人才纳入紧缺人才目录

### 政策影响

- 正面：政策持续加码，国产替代加速
- 风险：先进设备出口管制可能升级

## 八、投资机会与风险

### 投资机会

1. **AI算力芯片**：GPU、HBM、CPO等AI基础设施需求爆发
2. **国产设备**：北方华创、中微公司、拓荆科技等加速替代
3. **先进封装**：Chiplet技术路线带来的产业机会
4. **第三代半导体**：SiC在新能源汽车、光伏领域的渗透率提升

### 风险提示

1. **政策风险**：美国出口管制持续升级，先进设备获取受阻
2. **技术风险**：先进制程研发不及预期
3. **周期风险**：半导体行业周期性波动
4. **竞争风险**：海外龙头降价挤压国产厂商利润

### 建议关注方向

- 设备：北方华创、中微公司
- 封测：长电科技、华天科技
- 设计：兆易创新、澜起科技
- 材料：华特气体、江化微

## 九、总结与展望

### 核心观点

1. 半导体是数字经济的基础设施，AI时代需求持续爆发
2. 国产替代是长期主线，政策+资本持续加码
3. 当前处于库存周期底部，关注AI算力、车规芯片等高景气赛道
4. 先进制程差距仍存，但成熟制程国产化率快速提升

### 关键观测指标

- AI芯片出货量（英伟达、AMD订单）
- 半导体设备招标数据
- 存储芯片价格走势
- 中芯国际先进制程进展

### 风险阈值

- 出口管制政策进一步升级
- 全球半导体需求大幅下滑
- 国产替代进程低于预期
