# 恒瑞医药企业研究报告

## 一、摘要

**核心结论**

恒瑞医药是国内医药行业龙头，聚焦创新药研发与销售。公司拥有国内最丰富的创新药管线之一，覆盖肿瘤、自身免疫、糖尿病、疼痛等多个治疗领域。2024年以来，公司创新药收入占比持续提升，业绩恢复增长轨道。

**投资亮点**

- 创新药管线丰富，12款已上市创新药+90+在研项目
- 国际化进展加速，多款产品海外授权
- 销售网络覆盖全国，终端推广能力强
- 研发投入持续高强度，2024年研发费用率约25%

**风险提示**

- 集采压力持续，仿制药收入承压
- 创新药竞争加剧，价格下行压力
- 研发项目众多，临床失败风险
- 海外拓展不确定性

*（本报告不含买卖建议）*

## 三、行业分析

### 医药行业趋势

1. **创新药崛起**：在集采压力下，创新药成为行业增长的主要驱动力
2. **国际化加速**：国产创新药出海成为趋势，License-out案例增多
3. **差异化竞争**：同质化产品空间收窄，差异化创新成为核心竞争力

### 竞争格局

**国内创新药竞争态势：**

| 企业 | 优势领域 | 在研创新药数量 |
|------|----------|----------------|
| 恒瑞医药 | 肿瘤、疼痛、麻醉 | 90+ |
| 百济神州 | 肿瘤 | 50+ |
| 信达生物 | 肿瘤 | 30+ |
| 君实生物 | 肿瘤、免疫 | 30+ |
| 复星医药 | 肿瘤、疫苗 | 40+ |

**恒瑞医药竞争优势：**
- 肿瘤领域全覆盖，布局最全面
- 销售团队规模庞大，终端覆盖力强
- 现金流充沛，研发投入可持续
- 仿制药提供稳定现金流，支撑创新投入

**重点产品布局：**

- 自身免疫：IL-17A抑制剂、JAK抑制剂
- 糖尿病：SGLT-2抑制剂、DPP-4抑制剂
- 疼痛： MOR拮抗剂、长效局麻药

- 覆盖全国31个省、市、自治区

- 强大的品牌影响力

### 3. 产业化能力

- 符合GMP标准的现代化生产基地
- 关键原料药自主生产
- 成本控制能力突出
- 质量管理体系完善

### 4. 国际化能力

**海外进展：**
- 多个创新药在美国、欧洲、日本开展临床
- 卡瑞利珠单抗、吡咯替尼等多项海外授权达成
- 海外研发中心布局美国、欧洲
- 目标：未来5年海外收入占比达到20%

## 七、风险提示

### 1. 经营风险

- **研发风险**：创新药临床失败率高，在研项目存在失败可能
- **竞争风险**：PD-1等热门靶点竞争激烈，存在价格战风险
- **商业化风险**：新药上市后销售推广不及预期

### 2. 政策风险

- **集采风险**：仿制药持续面临集采降价压力
- **医保谈判**：创新药纳入医保可能面临大幅降价
- **审批风险**：新药审批进度可能低于预期

### 3. 财务风险

- **研发投入高**：高强度研发投入压缩短期利润
- **商誉风险**：并购整合效果存在不确定性

### 4. 国际化风险

- **地缘政治**：海外业务面临地缘政治风险
- **审批不确定性**：FDA/EMA审批存在不确定性
- **竞争加剧**：海外市场与全球巨头竞争

### 5. 行业风险

- **技术迭代**：新技术（如基因疗法）可能颠覆现有竞争格局
- **替代风险**：新型治疗手段可能替代传统药物

---

## 八、总结

### 核心观测点

1. **创新药收入占比**：持续提升趋势，是估值重塑的关键
2. **海外授权进展**：国际化进度是重要催化剂
3. **在研管线进度**：临床III期及NDA阶段项目的进展
4. **集采影响**：仿制药集采续约价格变化

### 业绩兑现节点

- 2025年：创新药收入占比目标50%+
- 2026年：多个III期临床数据读出
- 2027年：国际化收入贡献开始显现

### 风险阈值

- 集采续约价格下降超过30%
- 重磅创新药临床失败
- 海外授权被退回
- 核心技术人员流失
